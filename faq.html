F.A.Q. Frequently Asked Questions

Why did Seg make this proposal?

I wanted to make an example of a game proposal for my portfolio which I would be comfortable to be read publicly. By using an existing IP, I can be more forthcoming with my design as I didn't have the franchise to loose in the first place.

What platforms would see this game released on?

Any platform. This project doesn't need anything exotic to run and the controls can be translated to a gamepad, touch interface, and keyboard & mouse. The game is about the narrative content and most devices should be able to run it. Therefor it's more about the production bandwidth and business decisions.

But it won't work very well on a Coleco. ;)
